# Linux System Basics Lab

## What this repo is

This repo is my hands-on Linux learning lab.

I use this space to practice Linux fundamentals, document what I learn, and show my progress as I build stronger IT support, systems administration, and troubleshooting skills.

## What I am practicing

- Basic Linux commands
- Navigating the file system
- Creating, editing, moving, and deleting files
- Working with directories
- Viewing system information
- Checking logs
- Managing services with `systemctl`
- Practicing terminal workflows
- Building clean documentation habits

## Why this matters

Linux is widely used in IT, cloud, cybersecurity, servers, and infrastructure.

This lab helps me build the foundation needed to work with real systems, troubleshoot issues, and understand how Linux environments operate.
